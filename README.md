# PI_V_TIME-19 "PrimeStudy"
Projeto Integrar 5 / Puc Campinas Engenharia de Software 5º semestre

## 📍 O Projeto
O objetivo desse projeto é otimizar os estudos, facilitando e saparando as partes mais importantes, fazendo assim o estudando ser mais eficiente. Com o upload de arquivos PDF, a plataforma usa inteligência artificial para processar o conteúdo e gerar automaticamente:
* **Resumos Estruturados**
* **Tópicos Principais**
* **Flashcards e Perguntas**
